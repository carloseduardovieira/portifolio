# Default Tokens

This document includes both the `ColorTokensComponent` and the `SpacingTokensComponent`.

## Color Tokens

The `ColorTokensComponent` displays a table of color tokens. Each token includes a name, value, and a preview of the color.

## Spacing Tokens

The `SpacingTokensComponent` displays a table of spacing tokens (padding and margins). Each token includes a name, value, and a preview of the size.
